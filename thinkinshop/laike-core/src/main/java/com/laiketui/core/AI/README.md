## 必读

### 在线翻译

免费翻译接口：https://api.mymemory.translated.net/get?q=Hello%20World!&langpair=en|zh-CN

en-GB表示英式英语，zh-CN表示简体中文，这是语言代码的一种表示方式，常见的语言代码组合还有很多，以下是部分示例：
欧洲语言组合
en-US - en-GB：美式英语到英式英语，虽然都是英语，但在拼写、词汇、发音等方面有一些差异，比如美式英语中的 “color”，英式英语为
“colour”。
fr-FR - de-DE：法语（法国）到德语（德国），如法语 “Bonjour”（你好），德语为 “Guten Tag”。
es-ES - it-IT：西班牙语（西班牙）到意大利语，西班牙语 “hola”（你好），意大利语为 “ciao”。
亚洲语言组合
ja-JP - ko-KR：日语到韩语，日语 “こんにちは”（你好），韩语为 “안녕하세요”。
zh-TW - zh-CN：繁体中文（中国台湾地区）到简体中文，如繁体 “臺灣”，简体为 “台湾”。
hi-IN - ar-SA：印地语（印度）到阿拉伯语（沙特阿拉伯），印地语 “नमस्ते”（你好），阿拉伯语为 “مرحبا”。
其他语言组合
en-US - ru-RU：美式英语到俄语，英语 “hello”，俄语为 “привет”。
pt-BR - en-US：葡萄牙语（巴西）到美式英语，葡萄牙语 “olá”（你好），英语为 “hello”。
en-GB - th-TH：英式英语到泰语，英语 “goodbye”，泰语为 “ลาก่อน”。
