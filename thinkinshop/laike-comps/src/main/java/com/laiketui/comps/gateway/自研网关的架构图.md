自研网关的架构图：

## 网关架构图

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                              客户端                                          │
│   (小程序 / App / PC前端)                                                     │
└─────────────────────────────────┬───────────────────────────────────────────┘
                                  │
                                  ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                         CompsLaiKeApiController                             │
│                         =========================                           │
│  @RequestMapping("/gw")                                                     │
│                                                                             │
│  1. 获取客户端 IP ───► IP 黑名单检查 (Redis)                                   │
│  2. 解析请求参数 (module/app/action)                                          │
│  3. 生成 API Key                                                             │
└─────────────────────────────────┬───────────────────────────────────────────┘
                                  │
                                  ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                        服务发现 (从 Redis)                                    │
│  ┌────────────────────────────────────────────────────────────────────┐     │
│  │                    Redis                                           │     │
│  │  ┌─────────────────────────────────────────────────────────────┐   │     │
│  │  │ laike_api:user:list:v1     ← 服务注册                        │   │     │
│  │  │   {                                                         │   │     │
│  │  │     "module": "user", "action": "list", "version": "v1",    │   │     │
│  │  │     "url": "http://192.168.1.10:8080",                      │   │     │
│  │  │     "heartbeat": 1709999999999,                             │   │     │
│  │  │     "weight": 10                                            │   │     │
│  │  │   }                                                         │   │     │
│  │  └─────────────────────────────────────────────────────────────┘   │     │
│  │  ┌─────────────────────────────────────────────────────────────┐   │     │
│  │  │ laike_api:order:create:v1   ← 另一个服务实例                  │   │     │
│  │  │   { "url": "http://192.168.1.11:8080", ... }                │   │     │
│  │  └─────────────────────────────────────────────────────────────┘   │     │
│  └────────────────────────────────────────────────────────────────────┘     │
│                                  │                                          │
│                    ┌─────────────┴─────────────┐                            │
│                    │  loadAliveApis()          │                            │
│                    │  过滤心跳超时的实例          │                            │
│                    │  (30秒超时)                │                            │
│                    └───────────────────────────┘                            │
└─────────────────────────────────┬───────────────────────────────────────────┘
                                  │
                                  ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                        负载均衡 (LoadBalance)                                │
│                                                                             │
│   ┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐         │
│   │  轮询 (Round)   │    │  随机 (Random)   │    │  权重 (Weight)   │         │
│   │  实例1→实例2     │    │  随机选一个       │    │  实例1:10        │         │
│   └─────────────────┘    └─────────────────┘    │  实例2:5         │         │
│                                                 └─────────────────┘         │
└─────────────────────────────────┬───────────────────────────────────────────┘
                                  │
                                  ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                        请求转发 (GatewayService)                             │
│                                                                             │
│   gatewayService.invoke(api, dto)                                           │
│                                                                             │
│   ┌────────────────────────────────────────────────────────────────────┐    │
│   │  1. 构建 HTTP 请求                                                  │    │
│   │  2. 添加认证 Token                                                  │    │
│   │  3. 转发到目标服务 (http://192.168.1.10:8080/xxx)                    │    │
│   │  4. 接收响应                                                        │    │
│   │  5. 返回结果                                                        │    │
│   └────────────────────────────────────────────────────────────────────┘    │
└─────────────────────────────────┬───────────────────────────────────────────┘
                                  │
                                  ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                        返回结果给客户端                                        │
│                         { "code": "0", "data": {...} }                      │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 服务注册流程（后台服务）

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  用户服务    │     │  订单服务     │     │  商品服务    │
│ (192.168.1.10)│   │ (192.168.1.11)│   │ (192.168.1.12)│
└──────┬──────┘     └──────┬──────┘     └──────┬──────┘
       │                   │                   │
       │    定时向 Redis 注册 (每10秒)            │
       └───────────────────┼───────────────────┘
                           │
                           ▼
              ┌────────────────────────┐
              │        Redis           │
              │  laike_api:xxx:v1      │
              │  + heartbeat 时间戳     │
              └────────────────────────┘
```
