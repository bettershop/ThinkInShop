### Tomcat位置：tomcat启动无先后顺序

1、依赖laike-common模块的模块部署在tomcat1中，未购买的模块请忽略。

nacos中以下模块的配置中端口都要改成18001
```markdown
tocmat-21801 说明 端口21801必须和nacos配置中心配的端口一致 否则会找不到接口报404错误。                         
├── laike-admins pc管理后台【需部署，本地需启动】            
├── laike-apps 买家端模块 【需部署，本地需启动】        
├── laike-comps 公共组件【需部署，本地需启动】  
├── laike-plugins 营销插件接口【需部署，本地需启动】
```

5、spring-cloud-gateway 网关 JAR部署 【非必须】

一般只需要部署一次，保持持续运行。 【单独部署】

```markdow
jar包启动
├── 1、laike-cloud-gateway springcloud-gateway网关
```
