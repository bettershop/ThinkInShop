### 1、非插件接口实现模块结构
```markdown
laike-xxxx    #laike-模块名:需要部署的模块
├── pom.xml    #maven构建文件
├── src    #源码文件
│   ├── main
│   │   ├── java
│   │   │   └── com
│   │   │       └── laiketui
│   │   │           └── xxxx    #模块名
│   │   │               ├── LaikeXxxxApplication.java    #模块启动类
│   │   │               ├── common    #模块内通用文件夹
│   │   │               │   ├── consts    #模块内常量文件夹
│   │   │               │   └── uitls    #模块帮助类文件夹
│   │   │               ├── controller    #模块控制层代码
│   │   │               └── service    #模块服务层代码 
│   │   └── resources
│   │       ├── application.yml    #模块配置文件
│   │       ├── logback-spring.xml    #日志配置文件
│   │       ├── static    #静态资源
│   │       └── templates    #静态模版
│   └── test    #测试代码
```

### 5、项目目录说明
```markdown
thinkinshop #项目名
├── db_logs 数据库脚本变更日志文件
├── docs      
│ ├── defaultImgs 默认图片
│ ├── imgs 文档中的图片
│ ├── lib root授权包和其他有些可能不好下载jar包
│ ├── soft 系统软件
│ │ ├── nacos2.1.1
│ │ ├── seata1.6.1
│ │ ├── sentinal
│ │ ├── tomcat
│ │ ├── apache-zookeeper-3.7.1-bin.tar.gz
│ │ ├── jdk-8u211-linux-x64.tar.gz
├── 1、laike-admins PC端 【需部署，本地需启动】
├── laike-admins-api PC端接口声明
├── 2、laike-apps 买家端【需部署，本地需启动】
├── laike-apps-api 买家端接口声明    
├── 3、laike-comps 公共组件【需部署，本地需启动】【内含网关，必须】
├── laike-comps-api 公共组件接口声明
├── 4、laike-plugins 移动端【需部署，本地需启动】
├── laike-plugins-api 移动端接口声明
├── 5、laike-common 公共模块
├── laike-common-api 通用接口声明
├── 6、laike-core   公共配置 
├── 7、laike-domain  实体类等
├── 8、laike-cloud-gateway 网关【非必须】
└── pom.xml 依赖配置文件
```

目前需要部署的就是是有标记的5个服务，可以部署在一台机器，也可以部署到多台机器。
